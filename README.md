\# AI Career Assistant 



A FastAPI-based backend application that analyzes resumes and provides AI-powered insights.



\## Features

\- Resume upload (PDF/DOCX)

\- Text extraction

\- AI analysis using OpenAI

\- REST API endpoints



\## Tech Stack

\- Python

\- FastAPI

\- OpenAI API

\- Uvicorn



\## Run Locally

pip install -r requirements.txt  

python -m uvicorn main:app --reload
